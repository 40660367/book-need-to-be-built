# PyQt5环境安装及配置

## 打开终端

在右侧桌面环境空白处，右键打开`Open Terminal Here`

## 安装PyQt4环境
```shell
apt install python3-pyqt4 -y
```


## 运行第一个PyQt程序，Hello World!
```
python3 /share/lesson/pyqt5/helloworld.py
```
